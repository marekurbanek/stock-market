This is recruitment task for Future Processing.
To see live app please go to http://market-stock-fp.s3-website.eu-west-3.amazonaws.com
For some reason it works only in Chrome, so if you can't get it working please follow theese instructions:
1. git clone this repository
2. cd into root directory
3. npm install
4. cd client
5. npm install
6. npm start
7. Open another terminal
8. cd into root directory
9. node server.js
10. Go to localhost:3000 in the browser
11. Have fun trading!
